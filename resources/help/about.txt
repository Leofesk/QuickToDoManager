Description:

This application helps you collect all yours everyday tasks in one table.
Simple and comfortable way to managing your tasks.

Basic functional:

- Create new database.
- Open existing database.
- Add / edit / delete task.
- Change status.
- Tracking time to deadline and history, when task was created.
